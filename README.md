# 刑法抽卡

可選擇1抽或10抽來隨機抽出指定法規類型的條文。

可切換「日本」或「台灣」的法規。

可切換各法規來源地之「刑法」或「憲法」。

（備註：台灣之憲法包含增修條文，複製與顯示時也會標註來源）

## 開發

```bash
bun install
bun run dev
```

## 建置

```bash
bun run build
```
